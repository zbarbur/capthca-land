# CAPTHCA Duality App

This project is a dual-track identity protocol exploration.

## Project Structure
- `index.html`: The Duality Slider (Perspective Choice).
- `app/`: Shared component architecture (Target for development).
- `tracks/light/`: Content, assets, and manifesto for the Symbiotic path.
- `tracks/dark/`: Content, assets, and manifesto for the Adversarial path.
- `docs/`: Research, vision, and strategy documents.

## Design Philosophy
The app shares logic (Handshake, ZKP, Merkle proofs) but switches "Skins" based on a global theme class.
- Light: Solarpunk, Harmony, Blue/Gold.
- Dark: Cyberpunk, Adversarial, Green/Black.
